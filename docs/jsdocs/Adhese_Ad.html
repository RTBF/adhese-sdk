Class: Ad

Adhese#Ad

Ad object represents an internal advertisement that can be requested from the server. Both request and response are defined by this object. An ad can be found in the mainObject.ads array, through it's format code.

Constructor

new Ad(options) → {void}

Parameters:
Name Type Description
options object An object containing the available options for this Ad.
Properties
Name Type Description
write boolean if true the ad will be inserted into the DOM on creation, using Javascript's document.write function
location string An optional string that identifies this Ad's location. Beware, this will overwrite the global Adhese.location property for this Ad.
position string An optional string defining this Ad's position. The string will be added to Adhese.location to identifiy this Ad. Intended for using the same formatCode more than once in the same location.
